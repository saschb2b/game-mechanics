# Patterns Path of Exile 2 exemplifies

| Pattern | How PoE2 uses it |
|---|---|
| [`weapon-grants-skills`](/patterns/weapon-grants-skills) | Weapon archetype directly grants base skills. Major break from PoE1's "any skill on any weapon." |
| [`one-death-map`](/patterns/one-death-map) | One attempt per map node. Death ends the run; the node is consumed. |
| [`anti-screen-clear-combat`](/patterns/anti-screen-clear-combat) | Combat deliberately slowed to restore per-encounter meaning. Telegraph + dodge + commit. |
| [`enemy-intent-telegraph`](/patterns/enemy-intent-telegraph) | Bosses and elites visually telegraph attacks; the dodge roll is the response. |
| [`bonus-with-drawback`](/concepts/bonus-with-drawback) | Inherited from PoE1's keystones; ascendancy commits also locked-in (no in-character respec). |
| [`currency-as-crafting`](/patterns/currency-as-crafting) | Inherited from PoE1; rebalanced (Chaos/Exalted swap roles). |
| [`vision-driven-iteration`](/concepts/vision-driven-iteration) | Same philosophy as PoE1; new canvas, same Vision. |
| [`aspirational-crafting`](/patterns/aspirational-crafting) | Crafting bench gone, Recombinators in role of "next nerf candidate." Same cycle. |
| [`nested-progression-graph`](/patterns/nested-progression-graph) | Atlas as separate axis from character; weapon-set passives as tertiary axis. |
| [`parallel-products`](/patterns/parallel-products) | PoE2 ships alongside PoE1 rather than replacing it. Two games, two audiences. |
