# Concepts

Cross-game design patterns. Each concept page is **pattern-first** — a one-line lemma, a contrast table showing how multiple games instantiate the same idea, when to use, pitfalls.

> Concept pages are curated, written by hand. The auto-generated [patterns index](/patterns/) is the wider graph view of every pattern referenced anywhere across games.

## Available now

- [**Loadout as budget**](./loadout-as-budget) — Sparklite × Warframe × Slay the Spire. Spatial, numeric, and combinatorial flavors of the same idea.
- [**Handcrafted-PCG hybrid**](./handcrafted-pcg-hybrid) — All three games. Author at the unit level, randomize at the layout level.

## Stubs in the patterns index

Many other patterns have been tagged across games but don't have a curated concept page yet. Browse them at the [patterns index](/patterns/) — anything marked "stub" is a candidate for a future concept page.
